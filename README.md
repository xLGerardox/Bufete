# Bufete
Sistema con interfaz gráfica para gestionar clientes, expedientes, citas y tareas urgentes. Incluye ordenamientos (Bubble, Shell, Quick), búsquedas (binaria, hash) y recursividad. Diseñado para uso general de un bufete de abogadas

## Requisitos
- tkcalendar

## Cómo ejecutar
1. Instalar tkcalendar:
   pip install tkcalendar
2. Ejecutar:
   python Bufe.py

## Funciones principales
- Registro de clientes
- Registro de expedientes
- Agendar citas con calendario
- Tareas urgentes
- Búsqueda por ID
